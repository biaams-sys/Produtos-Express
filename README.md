# Produtos-Express

Apresentação:

Nesse projeto iremos criar uma API REST para produtos, onde podemos cadstrar, consultar, atualizar e excluir. Os dados estão estão nesse repositório então confira o código enquanto lê a descrição.

Descrição:

Oque é uma API REST?

É uma ferramenta que permite que diferentes programas se conectem. Ela usa as regras do REST, um estilo para construir sistemas que trocam informações.

--------------------------------------------

Com isso nesse projeto iremos criar um sistema de locação de acessórios da educação física. Usaremos o Xampp, Vscode e Insominia.

---------------------------------------------

Como instalar as dependências: 

1- Como instalar o XAMPP

Passos:
Acesse o site oficial do XAMPP (https://www.apachefriends.org/).
Baixe a versão do XAMPP compatível com seu computador.
Execute o instalador baixado.
Siga as instruções de instalação.
Após a instalação, abra o XAMPP.
Inicie os serviços Apache e MySQL para começar a usar o servidor local.

2- Como instalar o Visual Studio Code (VSCode)

Passos principais:
Acesse o site oficial do VSCode (https://code.visualstudio.com/).
Baixe o instalador para seu computador.
Execute o instalador e siga as ordens.
Depois de instalar, abra o VSCode.

3- Como instalar o Insomnia

Passos principais:
Acesse o site oficial do Insomnia (https://insomnia.rest/).
Baixe a versão adequada para seu computador.
Execute o instalador e siga as ordens.
Após instalar, abra o Insomnia.

Como executar o projeto:

Você deverá baixar os arquivos em uma pasta de sua escolha.
Dentro dos arquivos desse repositório você vai achar o arquivo Produtos Express.txt, nele você vai ter o script que irá usar no xampp. Então abara o xampp, aperte start na linha do Apache e do MySQL e espere as portas estarem verdes e ai aperte Admin na linha MySQL. Logo irá abrir uma pagina no seu navegador principal do localhost, nesse pagina vc deverpá apertar nos 3 tracinhos e  selecionar SQL e copiar o texto do arquivo txt.
